 ۳. دستورات نصب و اجرا

    
    

         قدم به قدم در ترمینال
        
        

            

                 وارد پوشه پروژه شوید
            

            

                cd مسیر/پوشه/EitaaReaction
                
            

        

        
        

            

                 نصب کتابخانه‌ها
            

            

                pip install -r requirements.txt
                
            

        

        
        

            

                 اجرای ربات
            

            

                python bot.py
                
            

        

        
        

            
            نکته: بار اول از شما کد تایید می‌پرسد. بعد از وارد کردن، حتماً y را بزنید تا دفعات بعد خودکار لاگین شود.

 ۵. راهنمای خطاها

    
    

         خطاهای رایج و راه حل
        

            
                

                    خطا
                    راه حل
                

            
            
                

                    ModuleNotFoundError
                    دستور pip install -r requirements.txt را دوباره بزنید
                

                

                    WebDriverException
                    کروم‌درایور را نصب کنید
                

                

                    Eitaa Is Down
                    اینترنت و فیلترشکن خود را چک کنید
                

                

                    Message not found
                    مطمئن شوید post_id درست است
                

                

                    Error editing message
                    ربات در کانال ادمین است؟ پیام کمتر از ۴۸ ساعت گذشته است؟
